# Lyubomyr Rudko Portfolio

Astro portfolio site for Lyubomyr Rudko.

## Commands

| Command | Action |
| :-- | :-- |
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview the production build |

## Notes

- Tailwind CSS is configured through the `@tailwindcss/vite` plugin.
- Public profile references currently use `@lyubmyr_rudko`.
- Homepage photos and project graphics are placeholder SVGs in `public/placeholders/`.
- The `/cv/` page renders placeholder CV content from `src/content/cv.md`.
